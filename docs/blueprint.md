# **App Name**: Shorts AI Script

## Core Features:

- Script Segmentation: Divide the input narration script into segments suitable for approximately 5-second video clips, based on sentence structure and natural pauses.
- Image Generation Prompts: For each script segment, generate a detailed image prompt that accurately reflects the content of the segment. The prompts should be suitable for use with AI image generation tools, so they include clear descriptions of subjects, actions, environments, and styles.
- Prompt Display: Display the image prompts alongside their corresponding script segments. The prompts are selectable, copyable, and/or downloadable.
- Guidance on AI Image tools: The LLM tool can make reasonable suggestions about what kind of AI tool might work for the selected prompts
- Prompt Editing and Regeneration: Allow users to edit the generated image prompts and regenerate images based on the modified prompts. This enables fine-tuning and customization of the visual content.
- Video Generation: If the images are satisfactory, allow users to generate a video from the images.

## Style Guidelines:

- Primary color: Soft sky blue (#87CEEB) to evoke a sense of calm and clarity, reminiscent of open skies and visual storytelling.
- Background color: Very light grey (#F5F5F5) to provide a clean, neutral backdrop that ensures readability of text and vibrancy of image prompts.
- Accent color: Warm orange (#FFB347) to highlight important actions such as copying prompts, indicating where the user should focus, or calling out LLM suggested tools.
- Body and headline font: 'PT Sans' for clear readability across various screen sizes, maintaining a friendly and accessible tone.
- Use a simple, consistent set of icons that represent various actions, such as copying text or downloading prompts, to facilitate intuitive use of the app.
- Design the layout so it shows segments and images as a cohesive set of content
- Use subtle animations to visually respond when the prompt is being segmented
